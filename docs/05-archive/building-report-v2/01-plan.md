# Plan: 기업 이전용 사옥 비교/상세 보고서 시스템

## 1. 개요 (Overview)
- **핵심 전환:** 단일 물건 정보 나열에서 **'다중 물건 비교 및 추천'** 중심으로 패러다임 전환.
- **목표:** 의사결정권자(대표)가 **3초 만에 최적의 대안을 파악**할 수 있는 2단 구조의 보고서 시스템 구축.
- **타겟 유저:**
  - **1차 (대표):** 비교 보고서를 통해 의사결정 (Why B is better than A).
  - **2차 (실무자):** 상세 보고서를 통해 계약 전 리스크 및 세부 스펙 검증.

## 2. 시스템 구조 (System Architecture)
```
[실무자 Input] 후보 물건 2~5개 주소 입력
       │
       ▼
┌───────────────────────────────┐
│ 1단계: 비교 보고서 (Decision) │
│ - 자동 추천 (Best Choice)     │
│ - 핵심 지표 1:1 비교          │
│ - 최종 설득 (누적 효과 분석)  │
└──────────────┬────────────────┘
               │ 클릭 시 상세 이동
               ▼
┌───────────────────────────────┐
│ 2단계: 상세 보고서 (Verify)   │
│ - 건축물대장, 실거래가        │
│ - 세부 비용/수익 시뮬레이션   │
│ - 법적 리스크 체크리스트      │
└───────────────────────────────┘
```

## 3. 주요 기능 (Core Features)

### 3.1 비교 보고서 (The Decision Maker)
- **자동 추천 시스템:** 유형별(임차/매매/투자) 가중치 알고리즘을 통해 1위 물건 자동 선정 및 추천 사유 생성.
- **스마트 비교표:** 
  - 항목별 최상위(Best) 데이터 자동 하이라이트 (⭐, 최저, 최대 태그).
  - A안 대비 B안의 증감액(Diff) 자동 계산 표시.
- **설득 논리 시각화:**
  - 임차: 3년 누적 비용 절감액.
  - 매매: 월 대출 상환액 vs 현 임대료 비교.
  - 투자: 은행 예금 이자 vs 예상 투자 수익 차액.

### 3.2 상세 보고서 (The Verifier)
- 기존 단일 보고서 기능 고도화.
- 건축물대장 데이터 시각화, 위치 기반 인프라 지도, 상세 금융 분석표 제공.

## 4. 데이터 모델 및 알고리즘

### 4.1 추천 점수 산정 로직 (Scoring)
- **임차(Lease):** 연간 절감액(30%) + 회수 기간(25%) + 전용률(15%) + 접근성(15%) + 안전도(15%)
- **매매(Purchase):** 월 부담 차액(25%) + 저평가 지수(25%) + 시세 상승률(20%) + 건물 등급(15%) + 안전도(15%)
- **투자(Invest):** Cap Rate(30%) + 공실 안정성(25%) + EXIT 수익(25%) + 현금 흐름(20%)

### 4.2 외부 데이터 연동
- **필수:** 국토부 건축물대장, 실거래가 API.
- **확장:** 공시지가, 상권 정보(소상공인시장진흥공단), 지하철 거리 계산(Map API).

## 5. UI/UX 전략
- **Comparison First:** 모바일에서는 가로 스와이프 카드로 물건 비교, PC에서는 스플릿 뷰(Split View)로 3개 물건 동시 비교.
- **Highlighting:** '이기는 항목'에 명확한 컬러/뱃지 부여.
- **Action-Oriented:** "B 물건 상세 보기" 버튼을 통해 자연스러운 뎁스 이동 유도.

## 6. 성공 기준
- 주소 3개 입력 시 비교 보고서 생성까지 5초 이내.
- 추천 물건이 실제 사용자(실무자)의 직관과 80% 이상 일치(알고리즘 튜닝 필요).
- 모바일에서 비교표 가독성 확보 (반응형 테이블/카드 전환).
