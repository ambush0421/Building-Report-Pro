# Building Report Pro V2 Plan Document

> Version: 1.0.0 | Created: 2026-02-11 | Status: Draft

## 1. Executive Summary
Building Report Pro V2는 주소 검색 한 번으로 완성되는 고품질 부동산 분석 보고서 서비스의 대대적인 업그레이드 버전입니다. 기존 13개 섹션에서 20개 섹션으로 대폭 세분화하며, 대출/레버리지 시뮬레이션, 주변 실거래가 조회, 건물 나이 분석 등 실무 투자 분석에 필수적인 기능들을 추가하여 전문가급 보고서를 제공합니다.

## 2. Goals and Objectives
- **실무 중심 분석 강화**: 단순 건물 정보 조회를 넘어 대출 시뮬레이션 및 수익률 분석 기능 강화.
- **데이터 다양성 확보**: 건축물대장 외에 토지정보(Vworld), 실거래가, 주변 편의시설 정보 통합.
- **사용자 경험 개선**: 호실별 상세 편집 및 전체 금액의 면적 비례 배분 기능으로 편집 편의성 극대화.
- **출력 최적화**: PDF 인쇄 최적화 스타일 적용으로 고품질 오프라인 보고서 제공.

## 3. Scope
### In Scope
- **시스템 초기화**: Vite + React + TS + Tailwind v4 + shadcn/ui 기반 재구축.
- **데이터 통합**: 건축물대장 API, 카카오 로컬/지도 API, Vworld 토지정보 API, 실거래가 API 연동.
- **금융 시뮬레이션**: LTV 기반 대출 시뮬레이션, NOI 기반 수익률 분석(Cap Rate, Leverage Yield 등).
- **호실 관리**: 집합건물 호실 선택 및 공실/임대 상태 관리, 금액 편집.
- **위치 기반 서비스**: 카카오맵 연동 건물 위치 및 주변 시설 시각화.
- **보고서 출력**: react-to-print를 이용한 PDF 출력 기능.

### Out of Scope
- **다크 모드**: 보고서 신뢰도 및 인쇄 품질을 위해 제외 (인쇄 최적화에 집중).
- **회원 시스템**: 초기 버전에서는 별도의 로그인 없이 주소 검색 기반으로 동작 (추후 확장 가능성).

## 4. Success Criteria
| Criterion | Metric | Target |
|-----------|--------|--------|
| 기능 완성도 | 전체 20개 섹션 구현 여부 | 100% 구현 |
| 데이터 정확성 | 공공데이터 및 계산 로직 정확도 | 100% 일치 (검증 완료) |
| UI/UX 품질 | 보고서 인쇄물 디자인 만족도 | 전문가 수준의 깔끔한 디자인 |
| 성능 | 주소 검색 후 데이터 로딩 시간 | 3초 이내 (병렬 호출 최적화) |

## 6. Risks
| Risk | Impact | Mitigation |
|------|--------|------------|
| 공공데이터 API 제한/지연 | High | 캐싱 전략 및 Promise.allSettled를 통한 부분 실패 허용 |
| CORS 이슈 | Medium | 필요 시 Cloudflare Workers 또는 프록시 서버 활용 |
| 카카오맵 SDK 연동 복잡성 | Low | 공식 문서 기반의 동적 로딩 구현 및 타입 정의 |
