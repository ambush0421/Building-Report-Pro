# AI Investment Scoring System Plan Document

> Version: 1.0.0 | Created: 2026-02-14 | Status: Draft

## 1. Executive Summary
전문가 수준의 20개 섹션 데이터를 일반 사용자도 쉽게 이해할 수 있도록 '투자 매력도 점수'로 변환합니다. 정량적 지표(수수료, 수익률)와 정성적 지표(입지, 노후도)를 결합한 고도화된 가중치 모델을 적용하여 객관적인 투자 가이드를 제공합니다.

## 2. Goals and Objectives
- **직관성 확보**: 복잡한 금융 지표를 단일 점수(0~100)로 시각화하여 빠른 판단 지원.
- **객관적 평가**: 개인의 주관을 배제하고 시장 데이터(실거래가, 지가추이) 기반의 점수 산출.
- **전략적 제언**: 점수대별(S, A, B, C 등급) 투자 전략 자동 생성.

## 3. Scope
### In Scope
- **가중치 모델 설계**: 수익성(40%), 안정성(30%), 입지(20%), 건물상태(10%) 비율의 스코어링 엔진.
- **실시간 점수 엔진**: 사용자가 매매가나 임대료 수정 시 즉시 점수가 재계산되는 로직.
- **비교 분석 통합**: 보고서 비교 기능과 연동하여 점수 기반의 순위 표시.
- **시각화 UI**: 대시보드 및 리포트 상단에 'AI Expert Grade' 배지 및 게이지 차트 노출.

### Out of Scope
- **딥러닝 학습**: 초기 버전은 정의된 알고리즘(Rule-based) 기반으로 동작하며, 추후 실제 매각 데이터를 통한 학습 모델로 확장.
- **예측 가격**: 미래 가격 예측 점수는 불확실성이 높으므로 제외하고 현재 가치에 집중.

## 4. Score Calculation Logic (Draft)
- **Profitability (40pt)**: Cap Rate 4% 이상 시 만점, Leveraged Yield 비중 반영.
- **Stability (30pt)**: DSCR 1.2 이상 여부, LTV 60% 이하 여부, 위반건축물 여부(감점).
- **Location (20pt)**: 지하철 거리(역세권 가점), 주변 인프라 밀집도.
- **Condition (10pt)**: 건물 연식(10년 이내 가점), 주차 및 승강기 설비.

## 5. Success Criteria
| Criterion | Metric | Target |
|-----------|--------|--------|
| 신뢰도 | 전문가 수동 평가 점수와의 상관관계 | 0.8 이상 |
| 반응 속도 | 데이터 변경 시 점수 갱신 시간 | 300ms 이내 |
| UI 만족도 | 사용자 테스트 시 점수 이해도 | '매우 쉬움' 90% 이상 |

## 6. Risks
| Risk | Impact | Mitigation |
|------|--------|------------|
| 지표 편향성 | High | 다양한 입지(수도권/지방) 데이터를 활용한 가중치 튜닝 |
| 데이터 결측 | Medium | 특정 데이터(지하철 등) 부재 시 해당 지표 가중치를 타 항목으로 배분하는 Fallback 로직 |
