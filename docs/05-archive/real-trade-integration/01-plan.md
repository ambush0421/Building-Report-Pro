# Plan: 실거래가 데이터 연동 (Real Trade Integration)

## 1. 개요 (Overview)
- **목적:** 건축물대장의 기본 정보와 국토교통부 실거래가 데이터를 결합하여, 보고서의 경제적 신뢰도를 극대화함.
- **배경:** 사용자는 현재의 건축물 스펙뿐만 아니라 해당 지역의 실제 거래 시세를 통해 투자 및 임차 타당성을 검토하고자 함.

## 2. 주요 기능 (Core Features)
1. **실거래가 API 연동:**
   - 국토부 상업업무용 및 매매 실거래가 API 호출 로직 구현.
   - 법정동, 시군구 코드를 기반으로 최근 1~3년 거래 내역 수집.
2. **유사 물건 매칭 (Smart Matching):**
   - 비교 대상 물건과 유사한 면적, 층수, 용도를 가진 거래 사례 우선 노출.
3. **시세 분석 데이터 생성:**
   - 평당 평균 매매가 산출.
   - 인근 지역 대비 해당 물건의 가격 경쟁력 지수 계산.
4. **시각화 (Visualization):**
   - 실거래가 추이 차트 (Time-series chart).
   - 지도상 거래 사례 마커 표시.

## 3. 데이터 모델
- **Input:** `sigunguCd`, `bjdongCd`, `yearMonth`
- **Output:** `dealAmount`, `area`, `floor`, `dealYear`, `dealMonth`

## 4. 성공 기준
- 주소 검색 시 해당 지역의 실거래가 정보가 3초 이내에 로딩됨.
- 비교 보고서에 '인근 실거래가' 항목이 추가되어 정량적 비교가 가능해짐.
