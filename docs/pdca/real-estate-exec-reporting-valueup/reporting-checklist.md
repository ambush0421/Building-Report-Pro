# 대표 보고 전 점검 체크리스트

## A. 내용 품질
- [ ] 대표 요약 1페이지 내 핵심 결론 3줄이 작성되었는가
- [ ] 필수 KPI 5종이 모두 기입되었는가
- [ ] 의사결정 요청 안건이 기한/담당자와 함께 작성되었는가
- [ ] 리스크 등급(High/Medium/Low)과 대응 액션이 연결되었는가
- [ ] 문장 규칙(`현상 -> 원인 -> 영향 -> 요청 의사결정`)을 지켰는가

## B. 데이터 신뢰성
- [ ] 기준일(As-of date)이 문서 상단에 명시되었는가
- [ ] KPI별 원천 시스템/집계 기준/추출 시각이 부록에 기재되었는가
- [ ] 전주 대비 증감 수치가 수식/집계와 일치하는가
- [ ] 분모 0 또는 누락값이 있는 KPI는 `N/A` 및 원인 코멘트가 있는가

## C. 보안/공유
- [ ] 개인정보(성명/연락처/상세주소)가 마스킹되었는가
- [ ] 외부 공유 시 내부 전용 코멘트가 제거되었는가
- [ ] 첨부파일 내 메타데이터(작성자명/경로)가 노출되지 않는가

## D. 보고 운영
- [ ] 작성자/검토자/결재자와 버전이 문서에 표시되었는가
- [ ] 마감시간 이전 대표 보고가 가능한 상태인가
- [ ] 결재 후 실행 액션 담당자에게 배정이 완료되었는가
